日常使用的forth程序庫
A forth lib for daliy usage.

希望能匯集常用的libs給大家寫代碼時候提供幫助，少做無用工。
We hope these libs can save your time when you coding with Forth on palm.

Gforth為測試平台（pc）， Dragon Forth為應用平台（Palm）。
Gforth is the testing platform on PC and Dragon Forth is the application platform on Palm Treo 650 mobile phone.
